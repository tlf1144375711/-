# 创建桌面快捷方式
```Shell
[Desktop Entry]
Type=Application
Name=Pycharm
GenericName=Pycharm3
Comment=Pycharm3:The Python IDE
Exec="/home/test1/Documents/pycharm/bin/pycharm.sh" %f
Icon=/home/test1/Documents/pycharm/bin/pycharm.png
Terminal=true
Categories=X-Application;
```
